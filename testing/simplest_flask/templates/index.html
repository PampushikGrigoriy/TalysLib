TalysLib web интерфейс (v.0.01)

Веб-интерфейс к библиотеке TalysLib , позволяющей выполнять расчеты различных яденых реакций с помощью программы Talys , а также выполнять сравнение полученных результатов с оцененными данными из ENDF

Сайт в разработке. Замечания и предложения по улучшению работы просьба писать на github по ссылке. Тест!.
